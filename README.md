Версия с++17, компилятор g++ 

Программа для преобразований математических выражений с одной переменной (x) и рациональными коеффициентами. 

Лексер, парсер реализованы при помощи декларативных языков Flex и Bison, остальной бекэнд (эмиттер) написан на С++. 

Реализованы операции сложения ("+"), вычитания ("-"), умножения ("*"), деления ("/"), возведения в степень ("^"). 

Есть поддержка унарных минусов и плюсов, числа ограничены типом long long. 

Реализовано два алгоритма Евклида для многочленов для подсчета НОД и сокращения дробей, а так же приведение старшего коеффициента многочлена в знаменателе к 1. 

Для всех операций во время вычислений добавлена проверка на переполнение. 

При переполнении во время вычисления НОД для сокращения дроби, сокращение не производится. 

Операция возведения в степень всегда предполагает наличие целого делителя (без переменных х).

Поддерживаемые грамматики (для неявного умножения): 5(x+1), x(x+1), (x+2)x

Недопустимые грамматики (для неявного умножения): (x+1)5, x x, 5 5

Числа не могут начинаться с 0!

Программа сначала показывает результат синтаксического разбора - очередь операторов и операндов (при успешном разборе), а затем выводит результат вычислений.
